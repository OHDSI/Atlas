Cohort exit criteria: For all persons who entered the cohort, there must be a specification of when each person exits the cohort. A person must exit the cohort at the end of the observation period spanning the qualifying initial event start date, but additional cohort exit criteria may be also considered.
Add a cohort exit criteria:
  • Based on a fixed time period relative to initial event start or end date
  • Based on the end of an era of persistent exposure to any drug within a defined concept set
The minimum data from amongst the selected cohort exit criteria occurring after the cohort entry date will be selected as the end date for the person's episode
