Cohort Pathway is defined as the process of generating an aggregated sequence of transitions between the Event Cohorts among those people in the Target Cohorts.

Analysis Settings

The following set of parameters will be used when performing the pathway analysis.

Combination Window: Window of time when two event cohorts need to overlap to be considered a combination.
Minimum cell count: Minimum number of subjects in the target cohort for a given event in order to be counted in the pathway.
Maximum path length: Maximum number of steps in a given pathway to be included in the sunburst plot.
